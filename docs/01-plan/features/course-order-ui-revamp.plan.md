## 1. Overview

### 1.1 Purpose

현재 강의 구매내역 페이지의 UI/UX를 개선하여 사용자가 구매한 강의(온라인/현장)의 주문 현황을 더 직관적으로 파악하고 관리할 수 있도록 한다.

### 1.2 Background

- bizschool은 "온라인 강의"와 "현장 강의(공개교육)" 두 가지 유형의 강의를 판매
- 현재 `CourseOrderSection.tsx`에 기본적인 테이블/카드 뷰가 구현되어 있으나 개선 여지가 있음
- 패스트캠퍼스, 인프런, Class101 등 주요 에듀테크 플랫폼의 구매내역 UI를 벤치마킹하여 사용자 경험 향상 필요
- "강의 구매내역"과 "도서 구매내역" 탭 분리 방식의 적절성 검증 필요

### 1.3 Related Documents

## 3. 탭 분리 vs 통합 분석

### 3.1 결론: 현재 탭 분리 방식 유지 권장

| 평가 기준 | 탭 분리 (현행) | 필터 통합 | 판정 |
|-----------|--------------|---------|------|
| 상품 성격 차이 | 강의(디지털)와 도서(물리)는 상태/컬럼이 완전히 다름 | 공통 컬럼이 거의 없음 | **탭 분리 우위** |
| 사용자 의도 | "강의 확인" vs "배송 확인"은 다른 의도 | 혼합 시 의도 파악 어려움 | **탭 분리 우위** |
| 필터 복잡도 | 각 탭에 맞는 필터만 표시 | 수강중+배송중 등 혼재 | **탭 분리 우위** |
| 액션 버튼 | 유형별 최적화된 액션 | 조건부 렌더링 복잡 | **탭 분리 우위** |
| NN/G 원칙 | "명확히 다른 성격 → 탭 적합" | - | **탭 분리 부합** |
| Class101 사례 | 구독/개별구매 별도 뷰 분리 | - | **탭 분리 패턴 확인** |

### 3.2 개선안: 탭 순서 및 레이블 최적화

현재: `"도서 구매내역" | "강의 구매내역"`

**개선 제안**: 사용 빈도와 비즈니스 중요도에 따라 순서 조정 가능
- 강의가 핵심 상품이면: `"강의 구매내역" | "도서 구매내역"` (강의 먼저)
- 또는 통합 카운트 배지 추가: `"강의 구매내역 (5)" | "도서 구매내역 (3)"`

## 6. UI 개편 상세 설계 방향

### 6.1 상태 요약 바 개선

```
현재:
┌─────────────────────────────────────────────────┐
│  주문건수(5)  결제대기(1)  결제완료(2)  수강중(1)  수강완료(1) │
└─────────────────────────────────────────────────┘

개선안:
┌──────────────────────────────────────────────────────────────┐
│  전체(8)  결제대기(1)  결제완료(2)  수강중(1)  수강완료(1)  환불(2)  취소(1) │
└──────────────────────────────────────────────────────────────┘
  → 환불/취소 상태 추가
  → 각 상태 클릭 시 해당 상태로 필터링 (인터랙티브)
```

### 6.2 필터 영역 개선

```
현재:
[6개월] [2026] [2025] [2024] [2023] [2022] [상세조회]

개선안:
[최근 1개월] [최근 3개월] [최근 6개월] [상세조회]
                                            ↓
                                    (모달에 연도별 선택 포함)

+ 강의유형 필터 칩 추가:
[전체] [온라인 강의] [현장 강의]
```

### 6.3 테이블 컬럼 개편

```
현재 (7컬럼):
| 날짜 | 강좌명 | 유형 | 결제금액 | 결제방식 | 결제여부 | 주문상태 |

개선안 (6컬럼):
| 주문일 | 강좌명 | 유형 | 결제금액 | 결제수단 | 상태 |
                                                  ↑ 결제여부+주문상태 통합

상태 배지 확장:
- 결제대기 (amber)
- 결제완료 (blue)
- 수강중 (purple)
- 수강완료 (emerald)
- 환불신청 (orange) ← NEW
- 환불완료 (gray) ← NEW
- 취소 (red) ← NEW
```

### 6.4 모바일 카드 개선

```
현재:
┌────────────────────────────┐
│ [수강중]          2026-01-15 │
│ 세무회계 실무 기초            │
│ 온라인 | 150,000원           │
│ 신용카드 · [결제완료]         │
└────────────────────────────┘

개선안:
┌────────────────────────────┐
│ [수강중]          2026-01-15 │
│ 세무회계 실무 기초            │
│ 온라인 강의                  │
│ ₩150,000                   │
│                    [상세보기] │
└────────────────────────────┘
  → 결제수단은 모바일에서 생략 (정보 밀도 축소)
  → 상세보기 버튼 추가
```

### 6.5 주문 상세 보기 (신규)

```
강좌명 클릭 또는 [상세보기] 클릭 시:

┌─────────────────────────────────────┐
│            주문 상세 정보              │
│─────────────────────────────────────│
│ 주문번호: CRS-2026-001              │
│ 주문일시: 2026-01-15 14:30          │
│─────────────────────────────────────│
│ 강좌명: 세무회계 실무 기초             │
│ 강의유형: 온라인 강의                 │
│ 수강기간: 2026-01-15 ~ 2026-07-15   │
│─────────────────────────────────────│
│ 결제금액: ₩150,000                  │
│ 결제수단: 신용카드                    │
│ 결제상태: 결제완료                    │
│─────────────────────────────────────│
│ [영수증] [수강하러 가기] [환불 신청]    │
└─────────────────────────────────────┘
```
