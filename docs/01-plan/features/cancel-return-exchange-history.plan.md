# Plan: 취소/반품/교환/환불내역 메뉴

## 1. 개요

마이페이지에 "취소/반품/교환/환불내역" 탭을 새로 추가하여, 기존 구매내역과 분리된 별도 메뉴로 취소/반품/교환/환불 관련 내역을 관리한다.

## 2. 배경 및 목적

- 현재 구매내역(purchases) 탭 안에서 교환/반품 신청 위자드가 동작하고 있으나, 취소/반품/교환/환불 **내역 조회**를 위한 전용 페이지가 없음
- 쿠팡 등 주요 이커머스 플랫폼처럼 "취소/반품/교환/환불내역" 전용 메뉴를 분리하여 UX 개선
- 사용자가 자신의 모든 클레임 이력을 한 곳에서 확인하고 상세 정보를 조회할 수 있도록 함

## 3. 기능 범위

### 3.1 신규 탭 추가
- 마이페이지 5번째 탭: "취소/반품/교환/환불내역" (`tab=claims`)
- 사이드바 + 모바일 탭 메뉴에 동시 반영

### 3.2 취소/반품/교환/환불내역 목록 페이지
- **서브탭**: "취소/반품/교환" | "무통장환불" (2개 탭)
- 안내 문구: "취소/반품/교환 신청한 내역을 확인할 수 있습니다."
- 하단 링크: 고객센터 1:1문의 / 취소/반품 안내 링크
- **각 내역 카드 표시 정보**:
  - 접수일 / 주문일 / 주문번호
  - 상품 이미지, 상품명, 상품 상세, 수량, 가격
  - 진행 상태 (취소완료, 교환진행, 반품, 교환철회 등)
  - 환불 예정일 정보
  - 상세보기 버튼 (취소상세, 교환상세, 반품상세)

### 3.3 취소 상세 페이지
- 타이틀: "취소/반품/교환/환불내역 상세"
- 주문일, 주문번호
- 상품 테이블 (상품, 금액, 진행 상태)
- **상세정보 섹션**: 취소접수일자, 취소접수번호, 취소완료일
- **취소 사유 섹션**: 취소 사유 표시
- **환불안내 섹션**: 상품금액, 배송비, 반품비, 환불 수단, 환불 완료 금액
- 취소영수증 확인 버튼
- 목록 버튼

### 3.4 반품 상세 페이지
- 상품 테이블 (상품, 금액, 진행 상태 + 회수조회/반품철회 버튼)
- **상세정보 섹션**: 반품접수일자, 반품접수번호
- **상품 회수정보 섹션**: 회수인, 휴대폰, 연락처, 주소 / 반송장정보 (택배기사, 송장번호)
- **회수 예정일/요청사항**: 회수 예정일, 회수 요청사항 + 변경하기 링크
- **반품 사유**: 사유 표시
- **환불안내**: 상품금액, 배송비, 반품비, 환불 수단, 환불 예상금액
- 취소영수증 확인 버튼
- 목록 버튼

### 3.5 교환 상세 페이지
- 상품 테이블 (상품, 금액, 진행 상태 + 교환철회 버튼)
- **상세정보**: 교환접수일자, 교환접수번호
- **교환 상품 배송정보**: 배송 진행 상태, 배송운송장 정보
- **상품 회수정보**: 회수인, 휴대폰, 연락처, 주소 / 반송장정보
- **회수 예정일/요청사항**: 변경하기 모달
- **교환 사유**: 교환 사유 표시
- 목록 버튼

### 3.6 취소 신청 위자드 (구매내역에서 진입)
- **Step 1**: 상품 선택 (체크박스)
- **Step 2**: 취소 사유 선택 (라디오 - 3가지 사유)
  - 배송지를 잘못 입력함
  - 상품이 마음에 들지 않음 (단순변심)
  - 다른 상품 추가 후 재주문 예정
- **Step 3**: 해결방법 선택
  - "배송지 변경하고 주문 유지" (배송지 변경 버튼) — 특정 사유에만 표시
  - "취소" (환불 정보 확인)
- **환불 정보 확인**: 상품금액, 배송비, 반품비, 환불 예상금액, 환불 수단
- **확인 모달**: "취소 신청하시겠습니까?" + "장바구니 다시 담기" 체크박스
- **완료 페이지**: 취소 상품 정보, 환불 예정일, 환불 수단, 만족도 평가 별점, "신청내역 확인하기" / "쇼핑 계속하기" 버튼

### 3.7 취소영수증 조회/발급 모달
- 상품 취소 금액, 할인금액 차감, 배송비, 총 환불 금액
- 취소영수증 출력 링크
- 카드 취소영수증 발급 정보

### 3.8 회수 예정일/요청사항 변경 모달
- 회수 예정일 표시
- 회수 요청사항: 문 앞 / 경비실 / 그 외 장소 (라디오)
- 이전 돌아가기 / 확인 버튼

### 3.9 주문목록/주문상세 상태 연동
- 취소 완료 후 주문목록에 "취소완료" 상태 표시 + "취소 상세 보기" 버튼
- 반품 접수 후 주문목록에 "반품접수" 상태 표시 + "반품 상세 보기" 버튼
- 교환 중인 주문에 "교환 중" 상태 + "교환 내역 보기" 버튼
- 교환/환불 신청 불가 안내 모달 (배송완료 후 7일 경과)

## 4. 기술 범위

- **프레임워크**: Next.js 16 (App Router)
- **라우팅**: `/mypage?tab=claims` + 상세 뷰는 컴포넌트 내부 state로 관리
- **타입**: TypeScript 인터페이스 확장
- **데이터**: Mock 데이터 기반 (현재 프로젝트 패턴 유지)
- **스타일**: Tailwind CSS + CSS Variables (기존 디자인 시스템 유지)
