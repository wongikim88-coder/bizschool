# Plan: BIZSCHOOL 메인 브랜드 로고 이미지

## 개요
참고 이미지(다이아몬드 4방향 패턴 + 중앙 문자)를 기반으로 BIZSCHOOL 프로젝트에 어울리는 메인 브랜드 로고를 SVG로 제작하고, Header/Footer 및 파비콘에 적용한다.

## 참고 이미지 분석
- **구조**: 4개의 다이아몬드(마름모) 블록이 십자 패턴으로 배치
- **색상**: 녹색(좌상), 주황(좌하), 짙은색(우상), 청록(우하)
- **중앙**: 흰색 문자 "D"가 보임
- **인상**: 역동적, 현대적, 성장/혁신 이미지

## 브랜드 로고 설계 방향

### 형태
- 참고 이미지와 동일한 4-다이아몬드 십자 패턴 채택
- 중앙 문자를 "B" (BIZSCHOOL의 이니셜)로 변경
- 깔끔한 기하학적 SVG로 제작

### 색상 팔레트 (BIZSCHOOL 맞춤)
| 위치 | 참고 이미지 | BIZSCHOOL 적용 | 의미 |
|------|------------|---------------|------|
| 좌상 | 녹색 | `#6abf4b` (성장 그린) | 비즈니스 성장 |
| 좌하 | 주황 | `#f5a623` (에너지 옐로) | 열정, 창의 |
| 우상 | 짙은색 | `#282c34` (차콜 다크) | 신뢰, 전문성 |
| 우하 | 청록 | `#3db5b5` (이노베이션 틸) | 혁신, 미래 |
| 중앙 문자 | 흰색 D | 흰색 B | BIZSCHOOL 이니셜 |

### 적용 위치
1. **Header 로고**: "BIZSCHOOL" 텍스트 왼쪽에 아이콘으로 배치 (28×28px)
2. **Footer 로고**: 동일 아이콘 + 텍스트 조합
3. **파비콘**: `/public/favicon.ico` 교체
4. **OG 이미지**: 필요 시 확장 활용

## 산출물
- `/public/images/logo.svg` — 메인 SVG 로고
- `/public/images/logo-white.svg` — 흰색 버전 (Footer용, 필요 시)
- `/public/favicon.svg` — 파비콘용 SVG
- Header.tsx / Footer.tsx 로고 적용 수정

## 우선순위
- P0: SVG 로고 파일 제작
- P1: Header/Footer 적용
- P2: 파비콘 적용
