# Plan: 공개교육 페이지 (public-education)

## 1. 개요

더존비즈스쿨 참조 사이트(dzbizschool.net)의 "공개교육 > 월별교육일정" 페이지를 BIZSCHOOL 프로젝트에 새로 구현한다. 기존 프로젝트의 디자인 시스템(Tailwind CSS v4 + CSS 변수)을 활용하여 모던한 UI로 재구성한다.

**참조 URL**: `https://www.dzbizschool.net/new/index.php?page=sub&menu=biz0201&year=2026&month=4`

## 2. 주요 기능 (Features)

### F-01: 페이지 히어로 섹션
- 공개교육 페이지 상단에 히어로 배너 표시
- 페이지 제목: "공개교육"
- 부제: "해당 교육과정을 클릭하시면 교육내용 및 수강신청을 하실 수 있습니다"
- 기존 about 페이지와 동일한 히어로 스타일 적용

### F-02: 연도/월 선택 필터
- 연도 드롭다운: 2006~2027년 (현재 연도 기본 선택)
- 월 탭 바: 1월~12월 가로 탭 형태
- URL 쿼리 파라미터로 year/month 관리 (`/education?year=2026&month=4`)
- 현재 선택된 월 하이라이트 표시

### F-03: 카테고리별 교육과정 테이블
- 카테고리 헤더: 카테고리명 + 아이콘 (세무회계, 특강, 재산제세(이론) 등)
- 테이블 컬럼: 교육과정명, 교육일시, 교육시간, 교육비, 교수, 접수
- 각 카테고리마다 별도의 테이블 헤더 반복
- 카테고리별 색상 구분 (accent color)

### F-04: 교육과정 상세 데이터 표시
- 과정명: 링크 형태 (클릭 시 상세 페이지 이동)
- 교육일시: "2026-04-03 ~ 2026-04-11(매주 금토)" 형식
- 교육시간: "13:00 ~ 19:00 (4일 24시간)" 형식
- 교육비: "550,000원" 형식
- 교수: "고경희 세무사" 형식
- 접수: "수강신청" 버튼

### F-05: 수강신청 버튼
- 모집중일 때: 활성 상태 (primary 색상 버튼)
- 마감일 때: 비활성 상태 (gray 색상 + disabled)
- 클릭 시 수강신청 모달 또는 상세 페이지 이동

### F-06: 검색 기능
- 교육과정명 검색 입력 필드
- 검색어 입력 시 해당 월의 과정 필터링

### F-07: 반응형 디자인
- Desktop: 6컬럼 테이블 레이아웃
- Tablet: 테이블 축소 또는 카드 전환
- Mobile: 카드 레이아웃으로 변환

## 6. 구현 범위

### In Scope
- 공개교육 메인 페이지 (월별 교육일정)
- 연도/월 필터링 (클라이언트 사이드)
- 카테고리별 교육과정 테이블
- 반응형 디자인 (Desktop/Tablet/Mobile)
- Header 네비게이션에 메뉴 추가
- Mock 데이터 기반 (API 연동 없음)

### Out of Scope
- 교육과정 상세 페이지 (`/education/[id]`)
- 실제 수강신청 기능
- 검색 API 연동
- 서브메뉴 (월별교육일정 외 다른 하위 메뉴)

## 7. 기술 스택

- Next.js 16 App Router
- React 19 + TypeScript
- Tailwind CSS v4 (CSS 변수 사용: `bg-[var(--color-primary)]`)
- lucide-react 아이콘
- `useSearchParams` for URL 쿼리 관리

## 8. 참조 페이지 스크린샷

참조 사이트의 주요 UI 요소:
1. **상단**: 파란색 배너 "더존비즈스쿨 is Specialist Edu"
2. **탭**: "월별교육일정" 탭 (현재 1개만 있음)
3. **검색바**: 검색어 입력 필드 + 검색 버튼
4. **연도 선택**: 드롭다운 (2026 선택)
5. **월 탭**: 1월~12월 가로 나열, 현재 월(4월) 하이라이트
6. **교육과정 테이블**: 카테고리별 분리, 6컬럼 (과정명/일시/시간/비용/교수/접수)
7. **수강신청 버튼**: 주황색 배경, 흰색 텍스트
