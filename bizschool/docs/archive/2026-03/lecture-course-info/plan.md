# Plan: 강의 수업 정보 입력 (Step 4)

## 개요

- **Feature**: lecture-course-info
- **Phase**: Plan
- **작성일**: 2026-03-27

## 배경 및 목적

`/expert/upload` 페이지의 Step 4. 전 단계(Step 3)에서 업로드한 영상 파일을 기반으로 강의 제목, 강의 설명, 개별 수업 제목을 입력하는 페이지.

두 가지 입력 모드를 제공한다:
1. **AI 자동 생성**: 업로드된 영상들을 AI가 분석하여 강의 제목·설명·수업 제목을 자동 생성
2. **직접 입력**: 사용자가 모든 항목을 직접 입력

실시간 미리보기 패널로 수강생 관점의 강의 소개 페이지를 즉시 확인할 수 있다.

## 핵심 기능

### F1. 모드 전환 — 세그먼트 탭 (Segmented Control)
- "✦ AI 자동 생성" | "직접 입력" 두 탭
- 탭 전환 시 이미 입력된 값은 보존
- AI 탭 → 직접 입력 전환 후 다시 AI 탭으로 돌아올 때 기존 입력이 있으면 "덮어쓸까요?" 확인 다이얼로그

### F2. AI 자동 생성 모드
- "AI로 자동 생성" 버튼 클릭 → 로딩 상태 (스켈레톤 UI)
- 생성 완료 시 각 필드에 값이 채워지며 "AI 생성됨" 배지 표시
- 사용자가 수정하면 배지 제거 (편집 가능)
- AI 오류 시 인라인 에러 + 수동 입력 모드로 자동 전환
- **실제 AI 연동은 Fake/Mock 처리** (Anthropic API 연동은 별도 Phase)

### F3. 강의 기본 정보 폼
- 강의 제목 (input, 필수)
- 강의 소개 (textarea, 필수, 최대 500자, 글자 수 카운터)

### F4. 커리큘럼 섹션
- Step 3에서 업로드된 파일 목록을 기반으로 자동 생성
- 각 영상별 수업 제목 입력 (input, 필수)
- 각 항목: 순서 번호 + 파일명 + "AI 생성됨" 배지 (해당 시) + 수업 제목 input
- 미입력 항목은 카드 헤더에 회색 "미입력" 표시
- "다음" 클릭 시 빈 필드가 있으면 첫 번째 빈 필드로 스크롤 + focus

### F5. 실시간 미리보기 패널
- **FAB 버튼**: `fixed bottom-8 right-8`, 원형, Eye 아이콘
  - 패널 닫힘 상태: 흰색 배경 + 테두리 + 그림자
  - 패널 열림 상태: primary 색상 배경
- **패널**: `fixed right-0 top-0 h-full w-[380px]`, z-50
  - 열기: `translate-x-0` (우→좌 슬라이드인, duration-300)
  - 닫기: `translate-x-full`
  - 외부 클릭 시 닫힘 (투명 backdrop)
  - 패널 내용: 강의 제목, 강사명, 썸네일 플레이스홀더, 강의 소개, 커리큘럼 목록
  - 미입력 필드는 이탤릭 회색으로 표시

### F6. 유효성 검사 및 다음 버튼
- 강의 제목 + 모든 수업 제목 입력 완료 시에만 "다음" 버튼 활성화
- Step 4의 "다음"은 no-op (Step 5는 추후 구현)

## 검증 시나리오

1. expert 계정으로 로그인 → 영상 업로드(Step 3) → Step 4 이동 확인
2. AI 탭에서 "AI로 자동 생성" 클릭 → 로딩 → Mock 데이터 자동 입력 확인
3. AI 생성 후 개별 필드 수정 → "AI 생성됨" 배지 제거 확인
4. 직접 입력 탭 전환 → 데이터 보존 확인
5. 일부 수업 제목 미입력 → "다음" 비활성화 / 클릭 시 빈 필드 포커스 확인
6. Eye FAB 클릭 → 패널 슬라이드인 확인 (우→좌)
7. 미리보기 패널 내 내용이 폼 입력과 실시간 연동되는지 확인
8. 패널 외부 클릭 / X 버튼으로 닫힘 확인
9. "이전" 클릭 → Step 3 복귀 확인
