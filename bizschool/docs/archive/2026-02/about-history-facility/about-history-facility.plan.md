# Plan: about-history-facility

> About 페이지에 연혁(History) 및 강의장 소개(Facility) 섹션 추가

## 1. 개요

현재 About 페이지(`/about`)에 3개 섹션(Hero, 핵심 서비스, 배움·나눔·성장 Cycle)이 존재하며,
이 뒤에 **연혁** 섹션과 **강의장 소개** 섹션을 추가한다.

## 2. 목표

- 비즈스쿨의 역사(2001~2011)를 타임라인 형태로 표시
- 강의장 시설 정보와 실제 사진을 통해 교육 환경을 소개
- 기존 페이지 디자인 패턴(색상, 간격, 타이포)과 일관성 유지

## 3. 추가 섹션 상세

### 3-1. 연혁 섹션 (Section 4)

**배경**: 흰색 (기존 Section 2와 동일 패턴)
**레이아웃**: 세로 타임라인 (좌측 연도, 우측 내용)

**데이터**:

| 연도 | 주요 내용 |
|------|-----------|
| 2001 | 서울시 성동교육청에 기업부설 평생교육원 설립신고 |
| 2001 | 제1,2 교육장(멀티미디어실 70석) 개설 |
| 2001 | 더존 네오플러스 실무용 버전 설치 |
| 2001 | 12월부터 사업주 훈련 교육 실시 |
| 2008 | 노동부 기관평가 재직자·실업자 모두 B등급 |
| 2008 | 노동부 직업능력시설 인정(세무회계분야) |
| 2008 | 지방특강 개최(대전, 대구, 부산) |
| 2008 | 수강지원금 과정 실시 |
| 2008 | 신입사원 위탁과정 및 자격증 과정 운영 |
| 2009 | 제3교육장 개설(멀티미디어실 30석) |
| 2009 | 휴게실 등 편의시설 확대 |
| 2009 | 국세청 공무원 대상 위탁교육 실시 |
| 2010 | 제4강의장 개설(멀티미디어실 25석) |
| 2010 | 더존 아이플러스 프로그램 설치 및 교육 |
| 2011 | 노동부 기관평가 B등급(동부지역 내 최고점수) |

**디자인**:
- 세로 타임라인 라인: `border-left` 또는 중앙선 방식
- 연도 뱃지: primary 색상 원형 또는 라운드 태그
- 모바일: 단일 컬럼 타임라인

### 3-2. 강의장 소개 섹션 (Section 5)

**배경**: `bg-[var(--color-light-bg)]` (기존 Section 3과 동일 패턴)

**텍스트 내용**:
- 총 40석 규모의 독립 강의장
- 1인1PC 멀티미디어 교육 가능
- 인터넷 연결 가능한 PC 및 노트북 비치
- 더존 Smart A 프로그램(실무용 버전) 설치
- 양도코리아 프로그램 설치

**이미지**:
- `bzschool_img_1.jpg`: 강의장 전경 (빈 교실 + 수업 중 모습 합성 이미지)
- `bzschool_img_2.jpg`: 동일 구도의 추가 사진
- 저장 위치: `public/images/about/`

**레이아웃**:
- 상단: 제목 + 설명
- 하단: 이미지 2장을 그리드(2열) 배치
- 이미지 아래: 시설 특징 리스트 (아이콘 + 텍스트)

## 4. 수정 파일

| 파일 | 작업 |
|------|------|
| `src/app/about/page.tsx` | 연혁 섹션, 강의장 소개 섹션 추가 |
| `public/images/about/bzschool_img_1.jpg` | 강의장 이미지 복사 |
| `public/images/about/bzschool_img_2.jpg` | 강의장 이미지 복사 |

## 5. 기술 사항

- Tailwind CSS v4: `var()` 래퍼 사용 필수 (예: `bg-[var(--color-primary)]`)
- Next.js `Image` 컴포넌트 사용 (최적화)
- 반응형: 모바일 → 태블릿 → 데스크탑 대응
- 기존 디자인 토큰 재사용 (`--color-dark`, `--color-muted`, `--color-primary` 등)

## 6. 구현 순서

1. 강의장 이미지를 `public/images/about/`에 복사
2. 연혁 데이터 배열 정의
3. 연혁 섹션 UI 구현 (타임라인)
4. 강의장 소개 섹션 UI 구현 (이미지 + 시설 정보)
5. 반응형 확인
