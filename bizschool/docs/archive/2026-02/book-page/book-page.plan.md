# Plan: 도서 페이지 (book-page)

> Feature: 도서 목록 페이지 구현
> Created: 2026-02-26
> Status: Plan
> Reference: https://www.dzbizschool.net/new/index.php?page=sub&menu=biz0401

---

## 1. 개요

더존비즈스쿨(dzbizschool) 도서 판매 페이지의 UI/UX를 참조하여 BIZSCHOOL 프로젝트에 도서 목록 페이지(`/books`)를 구현한다. 한 페이지당 최대 20권의 도서를 표시하고, 하단에 페이지 번호(1, 2, 3, 4...) 페이지네이션을 제공한다.

## 2. 참조 사이트 UI/UX 분석

### 2.1 페이지 구조 (dzbizschool)

| 영역 | 설명 |
|------|------|
| **페이지 배너** | "ETAX BOOKS is Good contents" 텍스트 + 도서 이미지 배경 |
| **탭 네비게이션** | 전체도서 / 조건표 / 장바구니 (3개 탭) |
| **카테고리 필터** | 전체, 더존테크윌간, 광교이택스간 등 출판사별 필터 탭 |
| **검색바** | 드롭다운(책이름) + 검색 입력 + 검색 버튼 |
| **도서 목록** | 가로형(horizontal) 카드 리스트 |
| **페이지네이션** | 하단 번호 기반 페이지 이동 |

### 2.2 도서 카드 레이아웃 (가로형)

```
┌─────────────────────────────────────────────────────────────────────┐
│ ┌──────────┐  제목 (볼드, 큰 폰트)  │ 저자 │ 출판사 │ 발행일      │
│ │          │  정가 : 100,000원                     ┌──────────┐    │
│ │ 책 표지  │  할인가 : 90,000원 (10%할인)          │ 장바구니  │    │
│ │ 이미지   │  ┌─────────────────────────────┐      ├──────────┤    │
│ │          │  │ 책 소개 텍스트 (회색 박스)  │      │ 바로구매  │    │
│ │          │  └─────────────────────────────┘      └──────────┘    │
│ └──────────┘                                                       │
└─────────────────────────────────────────────────────────────────────┘
```

### 2.3 핵심 정보 필드

- **제목**: 도서명 (굵은 글씨)
- **저자**: 저자명
- **출판사**: 출판사명
- **발행일**: YYYY년 MM월 DD일
- **정가**: 원래 가격 (취소선)
- **할인가**: 할인된 가격 (빨간/주황색, 할인율 표시)
- **설명**: 도서 소개 텍스트 (회색 배경 박스)
- **품절 표시**: [품절] 태그 (빨간색)
- **버튼**: 장바구니 (초록), 바로구매 (초록)

## 3. 구현 범위

### 3.1 포함 (In Scope)

| # | 기능 | 설명 |
|---|------|------|
| 1 | **도서 목록 페이지** | `/books` 라우트, 가로형 도서 카드 리스트 |
| 2 | **페이지 배너** | 도서 페이지 상단 배너 이미지/텍스트 |

| 4 | **검색 기능** | 도서명/저자 검색 |
| 5 | **페이지네이션** | 페이지당 20권, 하단 번호(1,2,3,4...) 페이지 이동 |
| 6 | **도서 카드 (가로형)** | 표지 이미지 + 도서 정보 + 가격 + 버튼 |
| 7 | **반응형 디자인** | 모바일/태블릿/데스크톱 대응 |

## 5. 데이터 모델

### 5.1 기존 Book 타입 확장

```typescript
// src/types/index.ts
export interface Book {
  id: string;
  title: string;
  author: string;
  cover: string;
  price: number;
  originalPrice?: number;
  discountRate?: number;
  rating: number;
  reviewCount: number;
  badges?: Badge[];
  // 새로 추가
  publisher?: string;        // 출판사
  publishDate?: string;      // 발행일
  description?: string;      // 도서 소개
  category?: string;         // 카테고리
  isSoldOut?: boolean;       // 품절 여부
}
```

### 5.2 카테고리 목록

|----------|------|

| 마케팅 | 마케팅 관련 도서 |
| 재무회계 | 재무/회계 도서 |
| 자기계발 | 자기계발 도서 |

## 7. 페이지네이션 사양

| 항목 | 값 |
|------|------|
| 페이지당 도서 수 | 20권 |
| URL 파라미터 | `?page=1&category=all&search=` |
| 페이지 번호 표시 | `< 1 2 3 4 5 >` 형태 |
| 최대 표시 번호 | 5개 (현재 페이지 중심) |
| 이전/다음 버튼 | `<` `>` 화살표 |

## 8. 구현 순서

1. **Book 타입 확장** - `publisher`, `publishDate`, `description`, `category`, `isSoldOut` 추가
2. **도서 데이터 확장** - 최소 40권 이상의 샘플 데이터 작성 (페이지네이션 테스트용)
3. **Pagination 컴포넌트** - 재사용 가능한 페이지네이션 UI
4. **BookListCard 컴포넌트** - 가로형 도서 카드 (참조 사이트 레이아웃)
5. **CategoryFilter 컴포넌트** - 카테고리 필터 탭
6. **BookSearch 컴포넌트** - 검색바
7. **BookBanner 컴포넌트** - 페이지 상단 배너
8. **도서 목록 페이지** - `/books/page.tsx` 조립 및 URL 파라미터 연동
