# Plan: training-page-tabs

> 근로자 주도훈련 페이지 UI 개선 및 탭 분리

## 1. 개요

### 1.1 배경
현재 `/training` 페이지는 모든 콘텐츠(프로그램 안내, 과정소개, 교육일정)가 하나의 긴 스크롤 페이지로 구성되어 있다. 콘텐츠를 탭으로 분리하고, UI 일관성을 개선하여 사용성을 높인다.

### 1.2 목표
- 페이지를 **"중소기업 근로자 주도훈련"** / **"과정소개 및 교육일정"** 2개 탭으로 분리
- 4단계 프로세스 박스 크기 통일
- 필수 제출서류 항목 정리 (불필요 항목 삭제, 아이콘 변경)

## 2. 요구사항

### 2.1 탭 구조 변경
| 항목 | 설명 |
|------|------|
| **탭 1** | "중소기업 근로자 주도훈련" — Hero + ProgramGuide (현재 콘텐츠) |
| **탭 2** | "과정소개 및 교육일정" — CourseIntro + TrainingMonthSelector + CourseTable |
| **기본 탭** | 탭 1 ("중소기업 근로자 주도훈련") |
| **탭 위치** | Hero 섹션 하단, 콘텐츠 영역 상단 |

### 2.2 4단계 프로세스 박스 크기 통일
| 항목 | 설명 |
|------|------|
| **대상** | "교육 준비", "교육 신청", "교육참여 및 수료", "지원금 신청" 4개 박스 |
| **변경** | 너비/높이를 동일하게 통일 (가장 큰 카드 기준으로 `min-h` 적용) |
| **현재 상태** | `grid-cols-4`로 너비는 동일하나, 내용 길이에 따라 높이가 다름 |

### 2.3 필수 제출서류 수정
| 항목 | 변경 내용 |
|------|-----------|
| **삭제** | "근로자 훈련 참여 사업주확인서" 삭제 |
| **삭제** | "근로자 훈련 참여 훈련기관 확인서" 삭제 |
| **아이콘 변경** | "훈련비 납입 영수증", "훈련 수료증" 앞 숫자(3,4) → 초록색 체크(v) 아이콘으로 교체 |
| **결과** | 2개 항목만 남음: 훈련비 납입 영수증, 훈련 수료증 (초록 체크 아이콘) |

## 3. 영향 범위

### 3.1 수정 파일
| 파일 | 변경 내용 |
|------|-----------|
| `src/app/training/page.tsx` | 탭 UI 추가, 탭별 콘텐츠 분리 |
| `src/components/training/ProgramGuide.tsx` | 4단계 박스 높이 통일, 필수 제출서류 수정 |
| (신규) `src/components/training/TrainingTabs.tsx` | 탭 네비게이션 클라이언트 컴포넌트 |

### 3.2 변경 없는 파일
- `src/components/training/CourseIntro.tsx` — 그대로 유지
- `src/components/training/TrainingMonthSelector.tsx` — 그대로 유지
- `src/data/training.ts` — 데이터 변경 없음

## 4. 구현 순서

1. `TrainingTabs.tsx` 클라이언트 컴포넌트 생성 (탭 상태 관리)
2. `page.tsx` 수정: Hero 아래에 탭 컴포넌트 배치, 탭별 콘텐츠 렌더링
3. `ProgramGuide.tsx` 수정:
   - 4단계 프로세스 박스에 `min-h` 적용하여 높이 통일
   - `requiredDocs` 배열에서 2개 항목 삭제
   - 남은 2개 항목의 숫자 아이콘 → 초록색 CheckCircle 아이콘으로 교체

## 5. 기술 참고사항

- Tailwind CSS v4: `bg-[var(--color-xxx)]` 형식 사용 필수
- 탭 상태는 URL query param 대신 클라이언트 state로 관리 (간단한 UI 전환)
- Hero 섹션은 탭 외부에 고정 (탭 전환 시에도 항상 표시)
