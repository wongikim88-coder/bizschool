# Plan: 커뮤니티 페이지 (community-page)

> Feature: 커뮤니티 페이지 구현 (홈/강의질문/상담사례/소통공간)
> Created: 2026-02-27
> Status: Plan
> Reference: https://www.inflearn.com/community, https://fastcampus.co.kr/community

---

## 1. 개요

BIZSCHOOL 플랫폼에 커뮤니티 페이지(`/community`)를 구현한다. 인프런 커뮤니티의 전체 UI 레이아웃을 참고하고, 패스트캠퍼스 커뮤니티의 우측 패널(주간 활동 TOP 10) 구조를 반영한다. 상단 검색창은 기존 글로벌 SearchBar를 유지하며, 검색창 하위에 홈/강의질문/상담사례/소통공간 4개 탭 섹션을 구성한다.

### 1.1 대상 사용자

- 경리/회계 업무 종사 직장인
- BIZSCHOOL 강의 수강생
- AI 전문가 상담 이용자

### 1.2 핵심 가치

- 수강생 간 지식 공유 및 소통
- 강의 질문에 대한 전문가 답변 접근
- AI 전문가 상담 사례의 공개 아카이브
- 커뮤니티 활성화를 통한 플랫폼 리텐션 강화

## 2. 참조 사이트 UI/UX 분석

### 2.1 인프런 커뮤니티 (`inflearn.com/community`)

| 영역 | 설명 |
|------|------|
| **상단 배너** | 커뮤니티 소개 텍스트 + 그라데이션 배경 |
| **탭 네비게이션** | 수강평, 로드맵, 스터디, 팀 프로젝트, 질문&답변, 고민있어요, 블로그 |
| **검색바** | 520px 너비, 둥근 모서리(50px radius), 중앙 배치 |
| **게시물 리스트** | 카드형 리스트, hover 시 배경 변경(#f1f3f5) |
| **레이아웃** | max-width 1440px, 반응형 그리드 |

### 2.2 패스트캠퍼스 커뮤니티 (`fastcampus.co.kr/community`)

| 영역 | 설명 |
|------|------|
| **탭 네비게이션** | 강의 질문, 스터디, AI 트렌드, 공지사항 |
| **메인 콘텐츠** | 게시물 피드 (카테고리 태그 + 제목 + 작성자 + 조회수/답변 수) |
| **우측 패널** | 주간 인기 질문 10개 + 주간 활동 TOP 10 유저 랭킹 |
| **게시물 카드** | 카테고리 분류, 제목, 닉네임(일부 마스킹), 답변 상태 표시 |

## 3. 구현 범위

### 3.1 포함 (In Scope)

| # | 기능 | 설명 |
|---|------|------|
| 1 | **커뮤니티 메인 페이지** | `/community` 라우트, 4개 탭 구조 |
| 2 | **탭 네비게이션** | 홈 / 강의질문 / 상담사례 / 소통공간 탭 전환 |
| 3 | **홈 탭** | 각 섹션의 인기 게시물 모아보기 + 우측 주간 활동 TOP 10 |
| 4 | **강의질문 탭** | 수강생 질문 + 전문가 답변 목록 |
| 5 | **상담사례 탭** | AI 상담 답변 + 전문가 검증 사례 목록 |
| 6 | **소통공간 탭** | 자유 게시글 + 댓글 목록 |
| 7 | **게시물 카드 컴포넌트** | 카테고리 태그, 제목, 작성자, 날짜, 조회수, 댓글/답변 수 |
| 8 | **주간 활동 TOP 10 패널** | 유저 랭킹 (닉네임, 활동 포인트, 순위 배지) |
| 9 | **반응형 디자인** | 모바일/태블릿/데스크톱 대응 |

### 3.2 제외 (Out of Scope)

- 게시글 작성/수정/삭제 기능 (버튼 UI만, 실제 동작 X)
- 댓글 작성 기능 (UI만, 실제 동작 X)
- 로그인/회원가입 연동
- 검색 필터링 실제 동작 (글로벌 SearchBar 유지, 커뮤니티 내부 검색 미구현)
- 게시글 상세 페이지 (추후 별도 feature)
- 알림/좋아요 기능
- 실시간 데이터 연동 (정적 mock 데이터 사용)

### 6.1 타입 정의

```typescript
// src/types/index.ts 에 추가

// 커뮤니티 게시물 공통 타입
export interface CommunityPost {
  id: string;
  title: string;
  author: string;
  category: string;
  createdAt: string;
  viewCount: number;
  commentCount: number;
}

// 강의질문 게시물
export interface CourseQuestion extends CommunityPost {
  type: "question";
  courseName: string;
  answerCount: number;
  isAnswered: boolean;
  expertAnswer?: string;
}

// 상담사례 게시물
export interface ConsultationCase extends CommunityPost {
  type: "consultation";
  aiAnswer: string;
  isVerified: boolean;
  verifiedBy?: string;
  helpfulCount: number;
}

// 소통공간 게시물
export interface DiscussionPost extends CommunityPost {
  type: "discussion";
  subCategory: "잡담" | "정보공유" | "질문" | "후기";
}

// 주간 활동 유저
export interface WeeklyActiveUser {
  rank: number;
  nickname: string;
  points: number;
  badge?: "gold" | "silver" | "bronze";
}

// 탭 타입
export type CommunityTab = "home" | "questions" | "cases" | "discussion";
```

## 8. 레이아웃 사양

### 8.1 데스크톱 (1200px+)

| 영역 | 사양 |
|------|------|
| 컨테이너 | max-width: 1200px, 중앙 정렬 |
| 홈 탭 메인 콘텐츠 | 약 70% 너비 (좌측) |
| 홈 탭 우측 패널 | 약 30% 너비 (우측, 홈에서만 표시) |
| 다른 탭 메인 콘텐츠 | 100% 너비 (우측 패널 없음) |
| 게시물 카드 | 가로 full-width, 리스트형 |
| 탭 네비게이션 | 가로 나열, 활성 탭 하단 border 강조 |

### 8.2 태블릿 (768px ~ 1199px)

- 우측 패널: 메인 콘텐츠 하단으로 이동 (1열 레이아웃)
- 게시물 카드: 동일 유지

### 8.3 모바일 (~767px)

- 탭: 가로 스크롤 가능
- 우측 패널: 메인 콘텐츠 하단 (접을 수 있는 아코디언 or 가로 스크롤)
- 게시물 카드: 간소화 (조회수 숨김 등)
